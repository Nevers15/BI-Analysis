# Карта навыков

### _Карта навыков разрабатывалась и продолжает расширять свой функционал для руководителей АО ОМК. Дашборд позволяет руководителю смотреть какие трудовые функции способен выполнять каждый сотрудник его команды._

__ЗАЧЕМ ЭТО РУКОВОДИТЕЛЮ:__

Руководитель эффективнее распределяет задачи между сотрудниками с учетом требований законодательства, при этом способствуя развитию команды через задачи.

• задачу можно дать тому, кто выполнит ее более качественно

• задачу можно дать для поддержания существующего навыка

• задачу можно дать для развития нового навыка 

• задачу нельзя дать тому, кто не имеет допуска

Профессиональное развитие сотрудников - это важный инструмент мотивации и удержания сотрудников в руках руководителя!
Руководитель может использовать Карту навыков для себя лично, демонстрировать при обходах или при аудитах.

# Укомплектованность наставниками

### _Отчет по укомплектованности наставниками был разработан как для руководителей Дивизионов / Подразделений / Участков, так и для самих рабочих. Дашборд позволяет руководителю отслеживать состояние укомплектованности своей Бизнес Единицы._

__ЗАЧЕМ ЭТО РУКОВОДИТЕЛЮ:__

Позволяет проводить мониторинг наличия наставников по каждому подразделению и получать детальную информацию по ним в разрезе Участка / Профессии / ФИО сотрудника. При отсутствии наставников в какой-либо профессии, руководитель участка может инициировать своего сотрудника на эту роль. Дашборд помогает повышать квалифицированность персонала компании.

# Отчет по входящему персоналу

### _Отчет по входящему персоналу был разработан для руководителей АО ОМК. Дашборд позволяет руководителю отслеживать состояние квалифицированности нанимаемого рабочего персонала._

__ЗАЧЕМ ЭТО директору:__

Это верхнеуровневая информационная панель с данными о процессе интеграции новых сотрудников в профессию.

__ЗАЧЕМ ЭТО РУКОВОДИТЕЛЮ:__

это консолидация информации о новых сотрудниках рабочих профессий в вашем подразделении (увидеть ситуацию, найти причины и зоны внимания, предпринять действия со своей стороны)

• позволяет увидеть объем новых сотрудников, нуждающихся в обучении по основной профессии

• проследить за процессом получения основной профессии новыми сотрудниками

• оценить эффективность удержания новых сотрудников
